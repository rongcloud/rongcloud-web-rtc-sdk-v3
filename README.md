### web-rtc-sdk

#### 开发指南

> 1、安装 [Node.js 10+](http://nodejs.cn/download/)

> 2、项目根目录执行 `npm install`

> 3、发布: npm run release

#### 目录结构

```
├── dist
├── release 发版记录
├── src 
  ├── 3rd
    └── adapter.js
  └── plugins
  └── providers
  └── enum.js
  └── event-emitter.js
  └── utils.js
├── README.md
├── package.json
```
