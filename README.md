### web-rtc-sdk

#### 开发指南

> 1、安装 [Node.js 6.3.0+](http://nodejs.cn/download/)

> 2、项目根目录执行 `npm install`

> 3、发布: npm run release

#### 目录结构

```
├── dist
├── release 发版记录
├── bugfix bug 记录
├── scripts 打包脚本
├── src 
  ├── 3rd
    └── adapter.js
  └── plugins
  └── engine.js 引擎
  └── rtc.js 对外暴露 API
├── test 单元测试
├── README.md
├── package.json
```
